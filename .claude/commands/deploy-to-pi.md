---

Deploye das GPS-Projekt auf den Raspberry Pi:

1. Prüfe ob alle Dateien committet sind
2. Erstelle ein Deployment-Paket (ohne .git, __pycache__, etc.)
3. Zeige die Schritte zum Transfer auf den Pi:
   - SCP-Befehle zum Kopieren
   - SSH-Befehle zur Installation
   - Systemd-Service-Setup (falls vorhanden)
4. Prüfe ob `scripts/deploy.sh` existiert, falls nicht, biete an es zu erstellen
5. Zeige wie man das Programm auf dem Pi startet

Gib klare, Copy-Paste-fähige Befehle aus.
